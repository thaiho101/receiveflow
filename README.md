# ReceiveFlow – Warehouse Receiving System

## 🚀 Overview
ReceiveFlow is a full-stack warehouse workflow system designed to automate order receiving and sorting processes. It replaces manual spreadsheet tracking with real-time validation and structured data processing.

## 🔥 Key Features
- Scan-based order validation and routing
- Real-time tracking of incoming items
- Automated sorting by organization and embroidery requirements
- Analytics dashboard with CSV export for reporting

## 📊 Impact
- Reduced order processing time by **7x** (1 week → 1 day)
- Eliminated manual sorting errors and data inconsistencies
- Improved operational visibility for warehouse staff

## 🛠 Tech Stack
- PHP (backend logic)
- MySQL (database)
- JavaScript
- HTML/CSS
- Google Charts

## ⚙️ How It Works
The system processes inconsistent source data into a structured SQL database.  
Warehouse staff scan incoming orders, and the system validates, classifies, and routes items based on predefined business logic.

## 📌 Notes
Originally designed from real warehouse workflows and later developed into a full web-based system.
